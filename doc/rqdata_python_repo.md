可获取深交所、上交所上市的各个期限的国债回购的日行情、分钟行情、tick 行情数据，具体调用方式请参考：[API-get_price](generic-api.md#rqdata-API-get_price).

## 银行间同业拆放利率 {#rqdata-shibor}

### get_interbank_offered_rate - 获取上海银行间同业拆放利率数据 {#rqdata-API-get_interbank_offered_rate}

```python
rqdatac.get_interbank_offered_rate(start_date=None, end_date=None, fields=None, source='Shibor')
```

获取上海银行间同业拆放利率数据

#### 参数 {#rqdata-API-get_interbank_offered_rate-params}

| 参数       | 类型                                                           | 说明                                                              |
| ---------- | -------------------------------------------------------------- | ----------------------------------------------------------------- |
| start_date | _int, str, datetime.date, datetime.datetime, pandas.Timestamp_ | 开始日期                                                          |
| end_date   | _int, str, datetime.date, datetime.datetime, pandas.Timestamp_ | 结束日期，不传入 start_date ,end_date 则 默认返回最近三个月的数据 |
| fields     | _str_, _str list_                                              | 字段名称，默认获取全部字段                                        |
| source     | _str_                                                          | 数据源，目前仅支持 Shibor                                         |

#### 返回 {#rqdata-API-get_interbank_offered_rate-return}

_pandas DataFrame_

| 返回   | 类型               | 说明                                                                                                                                   |
| ------ | ---------------   | -------------------------------------------------------------------------------------------------------------------------------------- |
| date   | _pandas.Timestamp_ | 日期                                                                                                                                   |
| fields | _float_           | 'ON'：隔夜<br/>'1W'：1 周<br/>'2W'：2 周 <br/>'1M'：1 个月 <br/>'3M'：3 个月周 <br/>'6M'：6 个月周 <br/>'9M'：9 个月周 <br/>'1M'：1 年 |
